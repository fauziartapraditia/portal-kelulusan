# Panduan Penggunaan Aplikasi Kelulusan SMK 5 AGUSTUS

Aplikasi ini dirancang untuk memudahkan sekolah dalam mengumumkan status kelulusan siswa secara online, transparan, dan efisien.

---

## 1. Fitur Utama & Tampilan Aplikasi

### A. Portal Siswa (Halaman Utama)
Halaman ini adalah pintu utama bagi siswa untuk mengecek kelulusan. Dilengkapi dengan hitung mundur otomatis dan formulir pengecekan.

![Halaman Utama Siswa](/kelulusan/assets/docs/home.png)

*   **Status Portal:** Menunjukkan apakah pengumuman sudah dibuka atau masih terkunci.
*   **Cek Kelulusan:** Masukkan NISN atau Nomor Peserta untuk melihat hasil.

### B. Akses Admin (Tersembunyi)
Untuk masuk ke panel admin, Anda tidak akan menemukan tombol "Login" di halaman utama demi keamanan.
*   **Cara Masuk:** Tekan tombol **`Ctrl + Alt + S`** secara bersamaan pada keyboard saat berada di halaman utama.
*   **Halaman Login:** Anda akan diarahkan ke halaman login berikut:

![Halaman Login Admin](/kelulusan/assets/docs/login.png)

> **Kredensial Default:**
> *   **Username:** `admin`
> *   **Password:** `admin123`

### C. Dashboard Admin
Setelah login, Anda akan melihat ringkasan data kelulusan siswa.

![Dashboard Admin](/kelulusan/assets/docs/dashboard.png)

*   **Statistik Cepat:** Jumlah total siswa, siswa lulus, tidak lulus, dan status portal.
*   **Akses Menu:** Navigasi cepat ke data siswa, nilai, dan pengaturan.

### D. Manajemen Data Siswa
Menu untuk mengelola daftar siswa yang akan diumumkan statusnya.

![Daftar Siswa](/kelulusan/assets/docs/siswa.png)

*   **Tambah Siswa:** Input data siswa baru satu per satu.
*   **Status Kelulusan:** Atur apakah siswa tersebut Lulus, Tidak Lulus, atau Ditunda.
*   **Akses SKL:** Admin dapat mengunci atau membuka akses download SKL untuk siswa tertentu.

### E. Manajemen Nilai (SKL)
Menu untuk menginput nilai mata pelajaran yang akan muncul di Surat Keterangan Lulus (SKL).

![Input Nilai](/kelulusan/assets/docs/nilai.png)

*   Setiap siswa memiliki inputan nilai yang dinamis sesuai dengan kurikulum yang diatur.

### F. Pengaturan Sistem
Konfigurasi utama aplikasi untuk menyesuaikan identitas sekolah dan jadwal.

![Pengaturan Sistem](/kelulusan/assets/docs/pengaturan.png)

*   **Identitas Sekolah:** Nama sekolah, alamat, dan logo.
*   **Jadwal Pengumuman:** Tentukan tanggal dan jam rilis pengumuman.
*   **Kepala Sekolah:** Nama dan NIP untuk tanda tangan di SKL.

---

## 2. Alur Penggunaan bagi Administrator

1.  **Login:** Gunakan shortcut `Ctrl + Alt + S`.
2.  **Konfigurasi:** Masuk ke menu **Pengaturan** untuk memastikan Nama Sekolah dan Jadwal Pengumuman sudah benar.
3.  **Data Siswa:** Masukkan daftar siswa melalui menu **Kelola Siswa**.
4.  **Input Nilai:** Lengkapi nilai siswa di menu **Kelola Nilai** agar SKL dapat dicetak dengan benar.
5.  **Cek Hasil:** Buka halaman beranda untuk memastikan hitung mundur berjalan sesuai jadwal.
